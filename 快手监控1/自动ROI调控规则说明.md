# 自动ROI调控规则说明

## 什么是自动ROI调控？

自动ROI调控是系统根据您的广告消耗情况，自动调整ROI（投资回报率）的功能。系统会定期检查您的广告消耗数据，并根据预设规则自动调整ROI值，以帮助您获得更好的投放效果。

## 调控规则

系统会根据您的广告消耗情况，按照以下规则自动调整ROI：

### 一、连续0消耗处理（最高优先级）

如果您的广告连续多轮消耗为0，系统会逐步降低ROI，以提高广告竞争力：

| 连续0消耗次数 | ROI调整幅度 |
|---------------|-------------|
| 2次 | 降低5个点 |
| 3次 | 降低10个点 |
| 4次 | 降低20个点 |
| 5次 | 降低30个点 |
| 6次及以上 | 降低40个点 |

### 二、不同消耗区间的处理

根据您的实际消耗情况，系统会自动调整ROI：

| 消耗范围（元） | ROI调整幅度 |
|----------------|-------------|
| 0 - 0.2 | 当前ROI ≤ 15：降低2个点<br>当前ROI > 15：降低4个点 |
| 0.2 - 0.3 | 当前ROI ≤ 15：降低1个点<br>当前ROI > 15：降低4个点 |
| 0.3 - 0.5 | 不调整 |
| 0.5 - 0.9 | 提高1个点 |
| 0.9 - 1.3 | 提高2个点 |
| 1.3 - 1.7 | 提高3个点 |
| 1.7 - 2.2 | 提高4个点 |
| 2.2 - 3 | 提高8个点 |
| 3 - 5 | 提高15个点 |
| 5 - 13 | 提高20个点 |
| > 13 | 提高40个点 |

## 边界限制

1. **最低ROI**：所有调整后的ROI不会低于2.66
2. **最高ROI**：所有调整后的ROI不会超过100
3. **智能投放跳过**：如果您使用智能投放（ROI显示为0），系统不会自动调整

## 智能投放说明

如果您的ROI显示为0，说明您正在使用智能投放模式，系统会跳过自动调整。智能投放由平台自动优化，无需手动干预。

## 调控执行频率

系统会定期执行自动调控，执行周期由管理员配置，通常为每轮数据更新后执行。

## 日志记录

系统会记录所有自动调整的历史记录，您可以在系统中查看：
- 调整时间
- 调整前后的ROI值
- 调整原因
- 调整结果

## 如何开启/关闭

您可以在系统设置中开启或关闭自动ROI调控功能。默认状态下，该功能为关闭状态。

## 常见问题

### Q：系统调整的ROI是否可以手动修改？
A：可以。您可以随时手动修改ROI值，系统会尊重您的手动调整，并在下次调控时基于您的手动调整值进行计算。

### Q：如果我对系统调整的ROI不满意，怎么办？
A：您可以手动修改ROI值，或者关闭自动调控功能，完全手动管理。

### Q：系统调整的ROI会立即生效吗？
A：是的，系统调整后，新的ROI值会立即应用到您的广告投放中。

### Q：我可以查看历史调整记录吗？
A：可以。系统会记录所有调整记录，您可以在系统中查看详细的历史调整信息。

## 总结

自动ROI调控功能旨在帮助您自动优化广告投放效果，根据实际消耗情况智能调整ROI值。您可以根据需要开启或关闭该功能，也可以随时手动干预调整结果。系统会记录所有调整历史，方便您查看和分析。
