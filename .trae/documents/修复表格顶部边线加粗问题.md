原因说明:
- Element Plus 在有 `border` 的表格上会通过伪元素 `::before` 绘制一条顶边线，同时表头单元格也可能有自身的顶部边框；两者叠加导致顶部边线看起来比其它线条粗 1px。

修复方案:
- 在 DataTable 组件的样式中禁用表格的顶边伪元素边框，保留表头自身的边线，避免叠加。
- 同时显式移除表头单元格的顶部边框，仅保留底部边线以保持层次。

具体修改:
- 在 `src/renderer/src/components/DataTable.vue` 的 `<style scoped>` 中添加:
  - `:deep(.el-table--border)::before { border-top-width: 0 !important; }`
  - `:deep(.el-table__header th) { border-top: 0 !important; }`

验证:
- 刷新后表格顶部边线与其他边线一致，不再出现加粗。
